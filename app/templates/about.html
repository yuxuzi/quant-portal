About the Integrated Dashboard

This application demonstrates the integration of multiple dashboard frameworks within a single Flask application.

Features

  • Flask as the main web framework
  • Plotly Dash for interactive data visualizations
  • Taipy for building data-driven web applications
  • Unified navigation system with Jinja templates
  • Responsive design using Bootstrap
  • Kerberos authentication for secure access

Technical Details

The application uses Flask as the primary web framework. Dash and Taipy are mounted within the Flask application, allowing seamless navigation between different dashboard tools while maintaining a consistent user interface and experience. Windows Kerberos authentication ensures secure access to all application resources.

Dashboard Frameworks

This application integrates:

  • Flask: Web framework
  • Dash: Interactive visualizations
  • Taipy: Data-driven applications
